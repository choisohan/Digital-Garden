# install and initial Set up 
[Document](https://github.com/goldfire/howler.js#documentation) 


1. Install on Your Project
```command
npm install howler
```

2. Import 
```js
import {Howl, Howler} from 'howler';
```

3. Load Sound
```js
var sound = new Howl({
  src: ['sound.webm', 'sound.mp3', 'sound.wav'],
  autoplay: true,
  loop: true,
  volume: 0.5,
  onend: function() {
    console.log('Finished!');
  }
});
```
